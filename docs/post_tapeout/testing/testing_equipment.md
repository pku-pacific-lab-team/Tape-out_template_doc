# 测试仪器使用指南

!!! warning "设备差异说明"
    不同品牌、不同型号、不同固件版本的仪器，其菜单路径和按键名称可能不同。请以**设备原厂用户手册**为准，本页仅提供通用操作原则与检查清单。

## 1. 通用安全与准备

上电前建议先完成以下检查：

1. 确认电源网连接正确（VDD/VSS 极性、线序、端子类型）。
2. 确认电压上限、电流限流值和功耗预算（避免误烧芯片）。
3. 确认测试板与仪器共地，避免浮地导致波形异常。
4. 建议先空载验证仪器输出，再连接负载。

!!! tip "建议流程"
    先低压、低限流启动，确认静态电流正常后再逐步升到目标工作点。

## 2. 可编程直流电源（DC Power Supply）

### 2.1 基本连接与输出策略

- 连接前先关闭输出（Output Off）。
- 先设置目标电压与限流，再开启输出（Output On）。
- 多路电源时，建议按依赖关系上电（先提供 IO 电压，再提供 Core 电压）。

### 2.2 常见问题

- 电流瞬间拉满：先断电，检查短路、焊接桥连、供电网络映射。
- 输出掉压：检查线材压降、接触电阻、限流设置是否过低。
- 波动较大：检查负载瞬态、去耦电容布局与接地路径。

## 3. 示波器（Oscilloscope）

### 3.1 探头与接地

- 优先使用短地弹簧，减少地线环路引入的噪声。
- 测量高速边沿时，优先关注探头带宽是否足够。

### 3.2 触发与采样

- 使用边沿触发（Edge Trigger）先稳定显示基础波形。
- 必要时使用单次触发（Single）捕捉异常瞬态。

### 3.3 常用观测项

- 时钟频率/占空比。
- 复位信号时序与释放顺序。
- 电源纹波与上电斜率（配合 AC 耦合或带宽限制功能）。
